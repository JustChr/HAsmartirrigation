# Branding assets

Source files for the Smart Irrigation mark and logo. Edit the SVGs here, then
regenerate the raster assets in the repo root / `docs/`.

| Source | Generated |
| --- | --- |
| `icon.svg` | `../icon.png` (256×256), `../icon@2x.png` (512×512), `../docs/favicon.ico` |
| `logo.svg` | `../logo.png` (600×180), `../logo@2x.png` (1200×360) |

## Concept

A water **droplet** with a **pulse / calculation line** — "smart, computed
watering, not guesswork." The pulse also nods to the live-status estimate the
integration shows between daily calculations.

- Water gradient: `#4ac7f5 → #0a7dd1`
- Wordmark: `SMART` in `#1aa7cf`, `Irrigation` in `#1583c0` (both legible on
  light and dark backgrounds)

All raster exports are transparent PNGs. The icon is **textless** by design
(the wordmark lives only in the logo lockup).
